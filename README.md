
## Installation

```st
Metacello new
	repository: 'github://StephanEggermont/ModularMonolith:master/src';
	baseline: 'ModularMonolith';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfModularMonolith asClass loadLepiter
```
